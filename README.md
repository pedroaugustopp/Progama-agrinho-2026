# 🌾 Agro Forte, Futuro Sustentável | Agrinho 2026

Site desenvolvido para o concurso **Agrinho 2026**, com o tema **"Agro Forte, Futuro Sustentável: o equilíbrio entre produção e sustentabilidade"**.  
O projeto explora práticas agrícolas conscientes, tecnologia no campo e o conceito de produzir respeitando o meio ambiente, tudo com um visual encantador inspirado no jogo **Stardew Valley**.

## 🎯 Objetivo

Apresentar de forma interativa e visualmente atrativa os desafios e soluções para uma agricultura produtiva e sustentável. O site é destinado a estudantes, educadores e todos que desejam compreender como aliar força produtiva à responsabilidade ambiental.

## 🧩 Funcionalidades

- **Design temático** com elementos pixel art, paleta de cores rurais e animações suaves.
- **Navegação suave** entre seções (Início, Tema, Práticas, Tecnologia, Quiz, Futuro).
- **Quiz interativo** com 5 perguntas sobre sustentabilidade no campo, com feedback instantâneo.
- **Contadores animados** de dados relevantes (agricultura familiar, redução de emissões, projeção populacional).
- **Revelação de elementos ao rolar** (scroll reveal).
- **Botão "voltar ao topo"** flutuante.
- **Totalmente responsivo** (desktop, tablet e mobile).
- **Código separado** em HTML, CSS e JavaScript puros, sem frameworks.

## 🚀 Tecnologias utilizadas

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações keyframes)
- JavaScript vanilla (manipulação DOM, Intersection Observer, contadores, quiz)
- Fontes Google: *Press Start 2P*, *Silkscreen*, *VT323*
- Emojis e elementos decorativos para reforçar o tema rural
